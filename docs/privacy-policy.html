ひとり飯アプリケーション

最終更新日: 2024年12月12日
📋 重要なお知らせ
本アプリは、ユーザーの皆様のプライバシーを最大限に尊重し、必要最小限のデータのみを取り扱います。このプライバシーポリシーでは、当アプリがどのような情報を収集し、どのように使用するかについて詳しく説明いたします。

1. 基本方針

「ひとり飯」（以下「本アプリ」）の開発・運営者（以下「当方」）は、本アプリをご利用いただくユーザーの皆様（以下「ユーザー」）のプライバシー保護を重要視し、個人情報の適切な管理に努めます。

2. 収集する情報

2.1 ユーザーが入力する情報

  • 食事名・料理名: ユーザーがうんちく生成のために入力する食べ物の名前
  • 利用目的: AIによるうんちく生成のため
  • 保存期間: 生成処理完了後、即座に削除（端末内・サーバー内ともに保存されません）

2.2 自動的に収集される情報

  • アプリ使用統計: 日次・月次の利用回数のみ
  • 保存場所: ユーザーの端末内のみ（SharedPreferences使用）
  • 保存期間: アプリ削除まで
  • 利用目的: レート制限機能の実現（1日20回、月600回の制限管理）

2.3 収集しない情報

  • 個人を特定可能な情報（氏名、住所、電話番号、メールアドレス等）
  • 端末の位置情報
  • 連絡先・写真・カメラ等へのアクセス
  • 他のアプリの使用状況
  • 生成されたうんちくの内容

3. 第三者サービスの利用

3.1 OpenAI API

本アプリは、うんちく生成のためにOpenAI社のGPT-4o APIを利用しています。

  • 送信データ: ユーザーが入力した食事名のみ
  • データの取り扱い: OpenAI社のプライバシーポリシーに従います
  • データ保持: OpenAI社の規定に従い、API利用完了後に削除されます
  • OpenAI プライバシーポリシー: https://openai.com/privacy

4. データの利用目的

  • AIによる食事のうんちく生成サービスの提供
  • アプリの利用制限管理（レート制限機能）
  • サービス品質の向上および改善
  • 不正利用の防止

5. データの共有・提供

当方は、以下の場合を除き、ユーザーの情報を第三者に共有・提供することはありません：

  • ユーザーの明確な同意がある場合
  • 法的な義務に基づく場合
  • ユーザーや公衆の安全を保護するために必要な場合
  • OpenAI APIへのうんちく生成リクエスト送信（食事名のみ）

6. データの保護

  • 適切な技術的・組織的セキュリティ対策を実施
  • 不正アクセス、紛失、破壊、改ざんからの保護
  • 最小限のデータのみ収集・利用
  • 定期的なセキュリティ見直し

7. ユーザーの権利

7.1 データの確認・削除

  • 利用統計の確認: アプリ内で現在の利用回数を確認可能
  • データの削除: アプリをアンインストールすることで、すべてのローカルデータが削除されます
  • 利用の停止: いつでもアプリの使用を停止できます

8. 子どもの保護

本アプリは全年齢対象ですが、13歳未満のお子様が利用される場合は、保護者の方の監督のもとでご利用ください。当方は意図的に13歳未満の児童から個人情報を収集することはありません。

9. 国際的なデータ転送

OpenAI APIの利用により、入力された食事名は米国のOpenAI社のサーバーで処理される場合があります。OpenAI社は適切なデータ保護措置を講じています。

10. ポリシーの変更

本プライバシーポリシーは、法的要件の変更やサービス改善のために更新される場合があります。重要な変更がある場合は、アプリ内またはサポートページでお知らせいたします。

11. 適用法律・管轄裁判所

  • 準拠法: 日本法
  • 管轄裁判所: 東京地方裁判所
  • 対応法規: 個人情報保護法、その他関連法令
